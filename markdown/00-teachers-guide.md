# Посібник викладача — ЄВІ English, 35 днів

## Формат іспиту (коротко)

**Англійська:** 30 завдань, ~45 хв — Matching (6) + Multiple choice (5) + Gapped text (19).

**ТЗНК:** 27 завдань, ~75 хв — gap-fill мікротексти (10) + multiple choice (23).

**Разом:** ~120 хв; мінімум «складено» — 5/30 (English) **і** 5/33 (ТЗНК).

**40%** формат + стратегії → **35%** Tier 1 grammar in context → **15%** лексика → **10%** ТЗНК

## Стратегії (вивчити до автоматизму)

### Task 1 — Matching (~8–10 хв)
1. Прочитати варіанти A–H, підкреслити key words
2. Швидко прочитати тексти 1–6 — main idea
3. Виключити 2 зайві
4. Перевірити відповідності

### Task 2 — Multiple choice (~10–12 хв)
1. Спочатку **питання**, потім текст
2. Шукати **paraphrase**, не буквальне слово
3. Eliminate 2 → обрати з 2

### Task 3 — Gapped text (~22–25 хв)
1. Грамматика чи лексика?
2. Вставити → **перечитати речення**
3. Skip and return (>90 сек на питання — далі)

## Граматика — пріоритети

**Tier 1 (тижні 1–3):** tenses, modals, articles, conditionals 0/1, passive, comparatives, pronouns, prepositions, word formation.

**Tier 2 (тижні 3–4):** 2nd conditional, reported speech, gerund/infinitive, relative clauses, used to.

**Tier 3 (якщо є час):** 3rd conditional, participle clauses.
