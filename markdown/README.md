# Підготовка до ЄВІ з англійської — пакет для викладача (35 днів, студент ~A2)

Практичні матеріали для проведення курсу підготовки до **ЄВІ** (магістратура). Студент орієнтовно на рівні **A2**; ціль — «складено» (100+) та робочий результат **140–155** рейтингових балів.

## Як користуватися

1. Прочитайте [посібник викладача](00-teachers-guide.md).
2. Роздрукуйте або надішліть студенту шаблони з [templates/](templates/).
3. Ведіть заняття за щотижневими планами (День 1 → Тиждень 5).
4. Фіксуйте прогрес у [KPI-трекері](templates/kpi-tracker.md).

## Інтенсивність

- **4–5 занять/тиждень** × 90 хв
- **45–60 хв домашньої роботи** щодня
- **3 повних mock** англійського блоку + **2 demo ТЗНК** + **1 full simulation 120 хв**
