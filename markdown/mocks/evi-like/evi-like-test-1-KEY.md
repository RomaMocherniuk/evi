## Teacher Notes — Common Errors at B1–B2

- **Gap (18):** "rather than" vs "than" — both seem comparative but "prioritise X rather than Y" requires "rather than"
- **Gap (19):** "besides" = in addition to; "beyond" = outside the scope of; confusing these is a B1–B2 hallmark error
- **Gap (22):** "compensate for" is a fixed collocation; "compensate" alone does not collocate with "off" or "over"
- **Gap (27):** gerund after "risk": "risk becoming" NOT "risk to become"
- **Task 1:** In matching tasks, identify the *main idea* of each paragraph first, then map to headings — avoid matching on individual words (e.g., "researchers" → D is tempting for text 3, but the main idea is language preservation)
