## Teacher Notes

- Gaps (12)–(19): focus on **basic grammar** (verb to be, present simple, articles, prepositions)
- Gaps (20)–(25): focus on **connectors** (and / but / so / because)
- Gaps (26)–(30): focus on **collocations and articles**
- Common A1–A2 errors: confusing "am/is/are", forgetting prepositions with time, using "a" before vowels
