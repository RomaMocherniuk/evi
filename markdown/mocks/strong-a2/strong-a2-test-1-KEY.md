## Teacher Notes

- Main grammar tested: **present perfect**, **future forms** (will / going to / present continuous), **comparatives**, **modal verbs**
- Key collocations to review: "look forward to", "at the top of the list", "read about", "let me know", "make/have a conversation"
- Common A2 errors: forgetting auxiliary "have" in present perfect, using "more cheap" instead of "cheaper"
