## Task 3

| Gap | Answer | Explanation |
|-----|--------|-------------|
| (12) | A — have | Present perfect: "platforms have transformed" (past-to-now effect) |
| (13) | A — depends | Present simple, singular: "this depends on…" |
| (14) | A — allows | Present simple fact: "social media allows" |
| (15) | A — otherwise | "that otherwise might never have received" — conditional adverb |
| (16) | D — have argued | Present perfect: describing ongoing/repeated claim |
| (17) | B — suggests | Present simple reporting verb, singular: "research suggests" |
| (18) | A — mostly | Adverb modifying verb: "share mostly the best version" |
| (19) | B — causing | Present participle: result clause in complex sentence |
| (20) | B — spreads | Present simple fact, singular: "false news spreads" |
| (21) | A — Once | Temporal conjunction: "once a story has been shared…" |
| (22) | A — should take | Modal + active verb: "companies should take more responsibility" |
| (23) | B — were enforced | Second conditional: "if stricter rules were enforced" |
| (24) | C — would be | Second conditional result: "harmful content would be removed" |
| (25) | A — Despite | Concessive preposition: "despite this, many people…" |
| (26) | B — could | "content that could otherwise be difficult to find" — hypothetical |
| (27) | A — actively | Adverb: "use platforms actively rather than passively" |
| (28) | A — more | Comparative: "more important than ever" |
| (29) | A — should learn | Modal for recommendation: "young people should learn" |
| (30) | A — developing | "responsibility for developing this skill" — gerund after preposition |

## Teacher Notes

- Main grammar tested: **second conditional** (gaps 23–24), **present perfect vs simple** (12, 16), **adverbs vs adjectives** (18, 27), **conjunctions / discourse markers** (21, 25), **gerunds after prepositions** (30)
- Key vocabulary: misinformation, viral, sensational, onboarding, distributed, brainstorming
- Common A2–B1 errors: using "despite + clause" (should be "although"), confusing "despite" and "although", forgetting gerund after "for"
