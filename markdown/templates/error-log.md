## Категорії помилок — довідник

| Категорія | Типові приклади |
|-----------|-----------------|
| tense | Past Simple vs Present Perfect; Continuous vs Simple |
| article | a/an/the/zero (school vs the school) |
| preposition | in/on/at; depend on; interested in |
| phrasal verb | give up, look after, find out |
| collocation | make a decision, heavy rain |
| word formation | success → successful; happy → happiness |
| relative clause | who/which/that/where |
| conditional | if + Present, will |
| passive | was built, is made |
| reading trap | те саме слово в тексті й відповіді |
| modal | must vs have to; should vs must |
